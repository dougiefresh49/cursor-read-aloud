Video game engines with AI support accelerate development through generative asset creation, code generation, and behavior scripting. Top engines include official AI suites (Unity, Unreal), specialized no-code builders (GDevelop), and next-generation AI-native platforms. [1, 2, 3, 4, 5]  
Top Game Engines with AI Features

• Unity: Features Unity AI, which includes agentic coding assistants, automated scene building, and generative tools for meshes and animations. These tools are built directly into the Unity Editor
.
• Unreal Engine: Offers advanced AI support through its behavior trees and plugin ecosystem. Developers can utilize AI plugins to assist with generating blueprints, procedural content (PCG), and animation behavior.
• GDevelop: An open-source, no-code engine with deeply integrated AI features. It allows creators to generate game mechanics, sprites, and dialogue directly from text prompts.
• Godot: Known for being lightweight and open-source. While it doesn't have a proprietary built-in AI, it features extensive third-party plugin support (e.g., using AI for pathfinding or integrating LLM assistants via its GDScript language).
• AI-Native Platforms: Emerging platforms like Rosebud AI and Wizard Genie function as specialized AI engines that allow you to generate complete games or modify project folders using natural language. [1, 2, 4, 6, 7, 8]

AI Capabilities in Game Development

1. Agentic Coding: AI can be used as a coding companion to generate scripts, refactor code, and fix bugs in real time.
2. Generative Assets: Fast-track asset pipelines by prompting AI to create concept art, textures, and 3D models.
3. Smart NPCs: Implement dynamic, conversational, and context-aware behavior trees for smarter non-player characters. [1, 4, 9, 10, 11]
